# NMP MoE Thermal ARIS Project

This project is an ARIS-ready workspace for studying thermal issues caused by irregular MoE routing in 3D-stacked NMP architectures.

## Focus

- architecture-level thermal behavior under irregular MoE expert routing
- coupling between routing skew, data placement, tier/bank activity, and hotspot formation
- thermal-aware mapping, scheduling, and architecture co-design
- HotSpot-based thermal evaluation as part of the validation path

## Directory Layout

- `CLAUDE.md`: project policy and pipeline dashboard
- `RESEARCH_BRIEF.md`: workflow 1 input brief
- `WORKFLOW1_RUNBOOK.md`: recommended ARIS startup commands
- `docs/research_contract.md`: active-idea contract
- `refine-logs/`: proposal, plan, and tracker files
- `experiments/hotspot/`: floorplan, config, and power-map placeholders
- `papers/`, `literature/`: local reading library
- `src/`: future modeling / analysis code

## First Step

Open this directory in your agent and run Workflow 1 after reading `RESEARCH_BRIEF.md` and `CLAUDE.md`.
